# Example Decision Matrix

This example illustrates how decision authority can be distributed.

| Decision Type | Decision Owner | Stakeholders | Escalation |
|---|---|---|---|
| Strategic investment | Executive leadership | Finance, strategy | Board |
| Program scope changes | Program leadership | Product, engineering | Executive sponsor |
| Technical architecture | Engineering leadership | Security, operations | CTO |
